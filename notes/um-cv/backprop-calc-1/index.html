TL;DR This article is a deep dive into the calculation of the gradient of a function defined on a matrix space. We will see how to define a differential form and how to calculate the gradient of a function defined on a matrix space. We will also see how to apply the chain rule in deep learning. We will see how to calculate the gradient of a neural network with a two-layer architecture.

We suppose that all functions are differentiable.

Introduction: Jacobian and gradient for vector functions

This section can be skipped if the reader is not familiar with differential geometry or the concept of the differential form.

From a higher point of view, the gradient, in finite dimensions or Hilbert Spaces, is the vectorial representation of a linear form in the dual space of the tangent space. See Riesz representation theorem.

Thus, the gradient here is supposed to have the same dimension as the domain of the function.

Consider a function f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm, the Jacobian matrix of fff is the matrix of all first-order partial derivatives of fff.

J(f)=[∂f1∂x1⋯∂f1∂xn⋮⋱⋮∂fm∂x1⋯∂fm∂xn]J(f) = \begin{bmatrix} \frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \\ \vdots & \ddots & \vdots \\ \frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n} \end{bmatrix} J(f)=​∂x1​∂f1​​⋮∂x1​∂fm​​​⋯⋱⋯​∂xn​∂f1​​⋮∂xn​∂fm​​​​

Consider a scalar function g:Rn→Rg: \mathbb{R}^n \to \mathbb{R}g:Rn→R, the gradient of ggg is the transpose of the Jacobian matrix of ggg.

∇g=J(g)⊤=[∂g∂x1⋯∂g∂xn]⊤=[∂g∂x1⋮∂g∂xn]\nabla g = J(g)^\top = \begin{bmatrix} \frac{\partial g}{\partial x_1} & \cdots & \frac{\partial g}{\partial x_n} \end{bmatrix}^\top = \begin{bmatrix} \frac{\partial g}{\partial x_1} \\ \vdots \\ \frac{\partial g}{\partial x_n} \end{bmatrix} ∇g=J(g)⊤=[∂x1​∂g​​⋯​∂xn​∂g​​]⊤=​∂x1​∂g​⋮∂xn​∂g​​​

The famous chain rule of differentiation can be expressed in terms of the Jacobian matrix.

Let f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm and g:Rm→Rpg: \mathbb{R}^m \to \mathbb{R}^pg:Rm→Rp, then the Jacobian matrix of the composition g∘fg \circ fg∘f is the product of the Jacobian matrices of fff and ggg.

J(g∘f)(x)=J(g)(f(x))⋅J(f)(x)J(g \circ f)(x) = J(g)(f(x)) \cdot J(f)(x) J(g∘f)(x)=J(g)(f(x))⋅J(f)(x)

Thus we have to be careful with the order of the Jacobian matrices when applying the chain rule with the gradient.

Let f:Rn→Rmf: \mathbb{R}^n \to \mathbb{R}^mf:Rn→Rm and g:Rm→Rg: \mathbb{R}^m \to \mathbb{R}g:Rm→R, then the gradient of the composition g∘fg \circ fg∘f is the product of the gradient of fff and the Jacobian matrix of ggg.

∇(g∘f)=(J(g)⋅J(f))⊤=J(f)⊤⋅J(g)⊤=(∇f)(x)⋅(∇g)(f(x))\nabla (g \circ f) = (J(g)\cdot J(f))^\top = J(f)^\top \cdot J(g)^\top = (\nabla f)(x) \cdot (\nabla g)(f(x)) ∇(g∘f)=(J(g)⋅J(f))⊤=J(f)⊤⋅J(g)⊤=(∇f)(x)⋅(∇g)(f(x))

be careful with the shape, here the jacobian matrix of ggg is 1×m1 \times m1×m and the gradient of fff is m×nm \times nm×n, whereas the gradient of ggg is m×1m \times 1m×1, and the gradient of g∘fg \circ fg∘f is n×1n \times 1n×1.

The most tedious part of the chain rule is to keep track of the shapes of the Jacobian matrices and the gradients.... especially when the function is defined on a matrix space or a tensor space.

The jacobian becomes somewhat pathological when the function is defined on a matrix space, because the gradient is always the same shape as the domain of the function (representation of the linear form in the dual space), for example when the function is defined on a matrix space.

Next we will see how to compute the gradient of a function defined on a matrix space.

General differential forms

Scalar function of a matrix variable

Suppose we have a function LLL that takes a tensor XXX as input and outputs a scalar.

  • LLL is a scalar function of a matrix variable XXX, for example with dimension (m×n)(m \times n)(m×n).

We want to define and compute

∂L∂Xor∇XL,\frac{\partial L}{\partial X} \quad\text{or}\quad \nabla_X L, ∂X∂L​or∇X​L,

where ∂L∂X\tfrac{\partial L}{\partial X}∂X∂L​ has the same shape of XXX (m×n)(m\times n)(m×n), whose element in the position (i,j)(i,j)(i,j) is ∂L∂Xi,j\tfrac{\partial L}{\partial X_{i,j}}∂Xi,j​∂L​.

[∂L∂X]i,j=∂L∂Xi,j.\left[\frac{\partial L}{\partial X}\right]_{i,j} = \frac{\partial L}{\partial X_{i,j}}. [∂X∂L​]i,j​=∂Xi,j​∂L​.

In higher order notation, we introduce the concept of differentials. If LLL is differentiable, then for a small change dXdXdX in the matrix XXX, we have

dL=∑i=1m∑j=1n∂L∂Xi,j dXi,jdL = \sum_{i=1}^m \sum_{j=1}^n \frac{\partial L}{\partial X_{i,j}} \, dX_{i,j} dL=i=1∑m​j=1∑n​∂Xi,j​∂L​dXi,j​

Or simply with Einstein notation:

dL=∂L∂Xi,j dXi,j.dL = \frac{\partial L}{\partial X_{i,j}} \, dX_{i,j}. dL=∂Xi,j​∂L​dXi,j​.

Tip

If the reader is familiar with the dual space, then he/she will recognize that the notion of small change is actually a differential form.

Important Frobenius product

We will use this following formula many times in the next sections.

We can write dLdLdL as a Frobenius product of the gradient of LLL and the differential of XXX.

dL=tr((∂L∂X)⊤ dX)dL = \mathrm{tr}\left(\left(\frac{\partial L}{\partial X}\right)^\top \, dX\right) dL=tr((∂X∂L​)⊤dX)

This is obtained by the fact that tr(ATB)=Ai,jBi,j\mathrm{tr}(A^TB)=A_{i,j}B_{i,j}tr(ATB)=Ai,j​Bi,j​.

Tensor function of a tensor variable

Suppose we have a function Y∈Rm×nY\in\mathbb{R}^{m\times n}Y∈Rm×n that takes a tensor X∈Rp×qX\in\mathbb{R}^{p\times q}X∈Rp×q as input and outputs a tensor.

We want to define and compute

∂Y∂Xor∇XY,\frac{\partial Y}{\partial X} \quad\text{or}\quad \nabla_X Y, ∂X∂Y​or∇X​Y,

where ∂Y∂X\tfrac{\partial Y}{\partial X}∂X∂Y​ is a tensor of shape (m×n×p×q)(m\times n\times p\times q)(m×n×p×q), whose element in the position (i,j,k,ℓ)(i,j,k,\ell)(i,j,k,ℓ) is ∂Yi,j∂Xk,ℓ\tfrac{\partial Y_{i,j}}{\partial X_{k,\ell}}∂Xk,ℓ​∂Yi,j​​.

[∂Y∂X]i,j,k,ℓ=∂Yi,j∂Xk,ℓ\left[\frac{\partial Y}{\partial X}\right]_{i,j,k,\ell} = \frac{\partial Y_{i,j}}{\partial X_{k,\ell}} [∂X∂Y​]i,j,k,ℓ​=∂Xk,ℓ​∂Yi,j​​

The general form of the chain rule for tensor functions

Let XXX be a matrix of shape (m×n)(m \times n)(m×n) and YYY be a matrix of shape (p×q)(p \times q)(p×q), and ZZZ be a scalar function of YYY. Then the chain rule for the tensor function ZZZ is:

∂Z∂Xi,j=∂Z∂Yk,ℓ⋅∂Yk,ℓ∂Xi,j\frac{\partial Z}{\partial X_{i,j}} = \frac{\partial Z}{\partial Y_{k,\ell}} \cdot \frac{\partial Y_{k,\ell}}{\partial X_{i,j}} ∂Xi,j​∂Z​=∂Yk,ℓ​∂Z​⋅∂Xi,j​∂Yk,ℓ​​

Until now, we have not seen how to calculate the gradient of a function defined on a matrix space and the backpropagation algorithm in deep learning. We will first recall the rules for calculating differential forms.

Given dfdfdf, we want to calculate ∂f∂X\frac{\partial f}{\partial X}∂X∂f​. We can write dfdfdf in the form of df=tr((∂f∂X)⊤ dX)df = \mathrm{tr}((\frac{\partial f}{\partial X})^\top \, dX)df=tr((∂X∂f​)⊤dX). To do this, we need some tricks to calculate the trace.

Tricks for calculating traces

Tricks
  1. Scalar rule: if aaa is a scalar, then tr(a)=a\mathrm{tr}(a) = atr(a)=a.
  2. Transpose rule: tr(A)=tr(A⊤)\mathrm{tr}(A) = \mathrm{tr}(A^\top)tr(A)=tr(A⊤).
  3. Linearity: tr(A+B)=tr(A)+tr(B)\mathrm{tr}(A + B) = \mathrm{tr}(A) + \mathrm{tr}(B)tr(A+B)=tr(A)+tr(B).
  4. Cyclic rule: tr(AB)=tr(BA)\mathrm{tr}(AB) = \mathrm{tr}(BA)tr(AB)=tr(BA).
  5. Product rule: tr(A⊤(B⊙C))=tr((A⊙B)⊤C)\mathrm{tr}(A^\top (B \odot C)) = \mathrm{tr}((A \odot B)^\top C)tr(A⊤(B⊙C))=tr((A⊙B)⊤C).
Frobenius Product

Let AAA and BBB be two matrices of the same shape. The Frobenius product of AAA and BBB is defined as

tr(A⊤B)=∑i,jAi,jBi,j\mathrm{tr}(A^\top B) = \sum_{i,j} A_{i,j}B_{i,j} tr(A⊤B)=i,j∑​Ai,j​Bi,j​

This is a generalization of the dot product for matrices.

Application in deep learning 1: Neural Network with two layers

  • The second term sums over jjj as −Pi,k∑j=1CYi,j-P_{i,k} \sum_{j=1}^C Y_{i,j}−Pi,k​∑j=1C​Yi,j​.

Since ∑j=1CYi,j=1\sum_{j=1}^C Y_{i,j} = 1∑j=1C​Yi,j​=1 (only one class is true for each sample):

dL=−1N∑i=1N∑k=1C(Yi,k−Pi,k)dZ2,i,kdL= -\frac{1}{N} \sum_{i=1}^N \sum_{k=1}^C (Y_{i,k} - P_{i,k}) dZ_{2,i,k} dL=−N1​i=1∑N​k=1∑C​(Yi,k​−Pi,k​)dZ2,i,k​

Then we can vectorize this calculation.

dL=−1Ntr((Y−P)⊤dZ2)=tr((P−YN)⊤dZ2)\begin{align*} dL &= -\frac{1}{N} \mathrm{tr}\left( (Y - P)^\top dZ_2 \right) \\ &= \mathrm{tr}\left(\left(\frac{P-Y}{N}\right)^\top dZ_2 \right) \\ \end{align*} dL​=−N1​tr((Y−P)⊤dZ2​)=tr((NP−Y​)⊤dZ2​)​

We have

∂L∂Z2=1N(P−Y)\frac{\partial L}{\partial Z_2} = \frac{1}{N}(P-Y) ∂Z2​∂L​=N1​(P−Y)

∂L/∂W2\partial L / \partial W_2∂L/∂W2​ and ∂L/∂b2\partial L / \partial b_2∂L/∂b2​

dZ2=d(H1W2+b2)=dH1W2+H1dW2+db2dZ_2 = d(H_1 W_2 + b_2) = dH_1 W_2 + H_1 dW_2 + db_2 dZ2​=d(H1​W2​+b2​)=dH1​W2​+H1​dW2​+db2​

Then

dL=tr((∂L∂Z2)⊤(dH1W2+H1dW2+db2))=tr((∂L∂Z2)⊤dH1W2)⏟dL1+tr((∂L∂Z2)⊤H1dW2)⏟dL2+tr((∂L∂Z2)⊤db2)⏟dL3\begin{align*} dL &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top \left( dH_1 W_2 + H_1 dW_2 + db_2 \right)\right) \\ &= \underbrace{\mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top dH_1 W_2 \right)}_{dL_1} \\ & \quad +\underbrace{\mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top H_1 dW_2 \right)}_{dL_2} + \underbrace{\mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top db_2 \right)}_{dL_3} \\ \end{align*} dL​=tr((∂Z2​∂L​)⊤(dH1​W2​+H1​dW2​+db2​))=dL1​tr((∂Z2​∂L​)⊤dH1​W2​)​​+dL2​tr((∂Z2​∂L​)⊤H1​dW2​)​​+dL3​tr((∂Z2​∂L​)⊤db2​)​​​

Since the first term and the third term are independent of dW2dW_2dW2​, we obtain

∂L∂W2=∂L2∂W2=H1⊤∂L∂Z2\frac{\partial L}{\partial W_2} = \frac{\partial L_2}{\partial W_2} = H_1^\top \frac{\partial L}{\partial Z_2} ∂W2​∂L​=∂W2​∂L2​​=H1⊤​∂Z2​∂L​

For the gradient of b2b_2b2​ we had gotta be careful. Since b2b2b2 is broadcasted, we have

dL3=tr((∂L∂Z2)⊤d(1Nb2))=tr((∂L∂Z2)⊤1Ndb2)=tr((1N⊤∂L∂Z2)⊤db2)\begin{align*} dL_3 &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top d(\mathbf{1}_{N}b_2) \right) \\ &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top \mathbf{1}_{N} db_2 \right) \\ &= \mathrm{tr}\left(\left( \mathbf{1}_{N}^\top\frac{\partial L}{\partial Z_2}\right)^\top db_2 \right) \\ \end{align*} dL3​​=tr((∂Z2​∂L​)⊤d(1N​b2​))=tr((∂Z2​∂L​)⊤1N​db2​)=tr((1N⊤​∂Z2​∂L​)⊤db2​)​

where 1N\mathbf{1}_{N}1N​ is a column vector of ones of shape (1,N)(1,N)(1,N).

Thus we have

∂L∂b2=1N⊤∂L∂Z2=1N∑i=1N(P[i,:]−Y[i,:])\frac{\partial L}{\partial b_2} = \mathbf{1}_{N}^\top \frac{\partial L}{\partial Z_2}=\frac{1}{N}\sum_{i=1}^N (P[i,:]-Y[i,:]) ∂b2​∂L​=1N⊤​∂Z2​∂L​=N1​i=1∑N​(P[i,:]−Y[i,:])

∂L/∂H1\partial L / \partial H_1∂L/∂H1​

Repeat the same calculation for dL1dL_1dL1​ and dH1dH_1dH1​.

dL1=tr((∂L∂Z2)⊤dH1W2)=tr(W2(∂L∂Z2)⊤dH1)=tr((∂L∂Z2W2⊤)⊤dH1)\begin{align*} dL_1 &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}\right)^\top dH_1 W_2 \right)\\ &= \mathrm{tr}\left(W_2\left(\frac{\partial L}{\partial Z_2}\right)^\top dH_1\right)\\ &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_2}W_2^\top\right)^\top dH_1\right)\\ \end{align*} dL1​​=tr((∂Z2​∂L​)⊤dH1​W2​)=tr(W2​(∂Z2​∂L​)⊤dH1​)=tr((∂Z2​∂L​W2⊤​)⊤dH1​)​

Thus we have

∂L∂H1=∂L∂Z2W2⊤\frac{\partial L}{\partial H_1} = \frac{\partial L}{\partial Z_2}W_2^\top ∂H1​∂L​=∂Z2​∂L​W2⊤​

∂L/∂Z1\partial L / \partial Z_1∂L/∂Z1​

Repeat

dL1=tr((∂L∂H1)⊤dH1)=tr((∂L∂H1)⊤(σ′(Z1)⊙dZ1))=tr((∂L∂H1⊙σ′(Z1))⊤dZ1)\begin{align*} dL_1 &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial H_1}\right)^\top dH_1 \right)\\ &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial H_1}\right)^\top \left( \sigma'(Z_1) \odot dZ_1 \right) \right) \\ &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial H_1}\odot\sigma'(Z_1)\right)^\top dZ_1 \right) \\ \end{align*} dL1​​=tr((∂H1​∂L​)⊤dH1​)=tr((∂H1​∂L​)⊤(σ′(Z1​)⊙dZ1​))=tr((∂H1​∂L​⊙σ′(Z1​))⊤dZ1​)​

Thus we have

∂L∂Z1=∂L∂H1⊙σ′(Z1)\frac{\partial L}{\partial Z_1} = \frac{\partial L}{\partial H_1}\odot\sigma'(Z_1) ∂Z1​∂L​=∂H1​∂L​⊙σ′(Z1​)

∂L/∂W1\partial L / \partial W_1∂L/∂W1​ and ∂L/∂b1\partial L / \partial b_1∂L/∂b1​

Repeat...

dL=tr((∂L∂Z1)⊤dZ1)=tr((∂L∂Z1)⊤(dXW1+XdW1+db1))=tr((∂L∂Z1)⊤XdW1)+tr((∂L∂Z1)⊤d(1Nb1))=tr((∂L∂Z1)⊤XdW1)+tr((∂L∂Z1)⊤1Ndb1)\begin{align*} dL &= \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_1}\right)^\top dZ_1 \right) \\ &=\mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_1}\right)^\top \left( dX W_1 + X dW_1 + db_1\right) \right) \\ &=\mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_1}\right)^\top XdW_1 \right) + \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_1}\right)^\top d(\mathbf{1}_{N} b_1) \right) \\ &=\mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_1}\right)^\top XdW_1 \right) + \mathrm{tr}\left(\left(\frac{\partial L}{\partial Z_1}\right)^\top \mathbf{1}_{N} db_1 \right) \end{align*} dL​=tr((∂Z1​∂L​)⊤dZ1​)=tr((∂Z1​∂L​)⊤(dXW1​+XdW1​+db1​))=tr((∂Z1​∂L​)⊤XdW1​)+tr((∂Z1​∂L​)⊤d(1N​b1​))=tr((∂Z1​∂L​)⊤XdW1​)+tr((∂Z1​∂L​)⊤1N​db1​)​

Then finally we have

∂L∂W1=X⊤∂L∂Z1 \frac{\partial L}{\partial W_1} = X^\top \frac{\partial L}{\partial Z_1} \ ∂W1​∂L​=X⊤∂Z1​∂L​ 

∂L∂b1=1N⊤∂L∂Z1\frac{\partial L}{\partial b_1} = \mathbf{1}_{N}^\top \frac{\partial L}{\partial Z_1} ∂b1​∂L​=1N⊤​∂Z1​∂L​

Summary

The regularized loss function is

L=−1Ntr(Y⊤log⁡P)+λ(W1,i,j2+W2,i,j2)L = -\frac{1}{N} \mathrm{tr}\left( Y^\top \log P \right) + \lambda \left( W_{1,i,j}^2 + W_{2,i,j}^2 \right) L=−N1​tr(Y⊤logP)+λ(W1,i,j2​+W2,i,j2​)

∂L∂Z2=1N(P−Y)\frac{\partial L}{\partial Z_2} = \frac{1}{N}(P-Y) ∂Z2​∂L​=N1​(P−Y)

∂L∂W2=H1⊤∂L∂Z2+2λW2\frac{\partial L}{\partial W_2} = H_1^\top \frac{\partial L}{\partial Z_2} + 2\lambda W_2 ∂W2​∂L​=H1⊤​∂Z2​∂L​+2λW2​

∂L∂b2=1N⊤∂L∂Z2\frac{\partial L}{\partial b_2} = \mathbf{1}_{N}^\top \frac{\partial L}{\partial Z_2} ∂b2​∂L​=1N⊤​∂Z2​∂L​

∂L∂H1=∂L∂Z2W2⊤\frac{\partial L}{\partial H_1} = \frac{\partial L}{\partial Z_2}W_2^\top ∂H1​∂L​=∂Z2​∂L​W2⊤​

∂L∂Z1=∂L∂H1⊙σ′(Z1)\frac{\partial L}{\partial Z_1} = \frac{\partial L}{\partial H_1}\odot\sigma'(Z_1) ∂Z1​∂L​=∂H1​∂L​⊙σ′(Z1​)

∂L∂W1=X⊤∂L∂Z1+2λW1\frac{\partial L}{\partial W_1} = X^\top \frac{\partial L}{\partial Z_1} + 2\lambda W_1 ∂W1​∂L​=X⊤∂Z1​∂L​+2λW1​

∂L∂b1=1N⊤∂L∂Z1\frac{\partial L}{\partial b_1} = \mathbf{1}_{N}^\top \frac{\partial L}{\partial Z_1} ∂b1​∂L​=1N⊤​∂Z1​∂L​

We can observe that the gradient of the layer deepens on the gradient of the next layer. This is the principle of backpropagation in deep learning.

# @Credits: UMich EECS 498-007 / 598-005 Deep Learning for Computer Vision
# Solution is my own implementation of the forward and backward pass of a two-layer neural network

def nn_forward_backward(
    params: Dict[str, torch.Tensor],
    X: torch.Tensor,
    y: Optional[torch.Tensor] = None,
    reg: float = 0.0
):
    """
    Compute the loss and gradients for a two layer fully connected neural
    network. When you implement loss and gradient, please don't forget to
    scale the losses/gradients by the batch size.

    Inputs: First two parameters (params, X) are same as nn_forward_pass
    - params: a dictionary of PyTorch Tensor that store the weights of a model.
      It should have following keys with shape
          W1: First layer weights; has shape (D, H)
          b1: First layer biases; has shape (H,)
          W2: Second layer weights; has shape (H, C)
          b2: Second layer biases; has shape (C,)
    - X: Input data of shape (N, D). Each X[i] is a training sample.
    - y: Vector of training labels. y[i] is the label for X[i], and each y[i] is
      an integer in the range 0 <= y[i] < C. This parameter is optional; if it
      is not passed then we only return scores, and if it is passed then we
      instead return the loss and gradients.
    - reg: Regularization strength.

    Returns:
    If y is None, return a tensor scores of shape (N, C) where scores[i, c] is
    the score for class c on input X[i].

    If y is not None, instead return a tuple of:
    - loss: Loss (data loss and regularization loss) for this batch of training
      samples.
    - grads: Dictionary mapping parameter names to gradients of those parameters
      with respect to the loss function; has the same keys as self.params.
    """
    # Unpack variables from the params dictionary
    W1, b1 = params["W1"], params["b1"]
    W2, b2 = params["W2"], params["b2"]
    N, D = X.shape

    scores, h1 = nn_forward_pass(params, X)
    # If the targets are not given then jump out, we're done
    if y is None:
        return scores

    # Compute the loss
    normalized_scores = scores - scores.max(dim=1, keepdim=True).values
    torch.exp(normalized_scores, out=normalized_scores)
    normalized_scores /= normalized_scores.sum(dim=1, keepdim=True)
    loss = torch.mean(-torch.log(normalized_scores[torch.arange(N), y]))
    loss += reg * (torch.sum(W1 * W1) + torch.sum(W2 * W2))
    
    # Backward pass: compute gradients
    grads = {}
    grad_z2 = normalized_scores
    grad_z2[torch.arange(N), y] -= 1  # For correct class, subtract 1
    grad_z2 /= N  # Scale by batch size

    grads['W2'] = h1.T @ grad_z2 + 2 * reg * W2  # Add regularization gradient
    grads['b2'] = grad_z2.sum(dim=0)  # Bias gradient (sum over N)

    # Compute gradient with respect to W1, b1
    grad_h1 = grad_z2 @ W2.T  # Backprop to first hidden layer
    grad_z1 = grad_h1 * (h1 > 0)  # Apply ReLU derivative

    grads['W1'] = X.T @ grad_z1 + 2 * reg * W1  # Add regularization gradient
    grads['b1'] = grad_z1.sum(dim=0)  # Bias gradient (sum over N)

    return loss, grads
